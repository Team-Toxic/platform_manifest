Team Toxic 7.0 Nougat WIP (AOSP)!
===========

To initialize your local repository using the Toxic 7.0 trees, use a command like this:
````bash
repo init -u git://github.com/Team-Toxic/platform_manifest.git -b android-7.0
```
Then to sync up:
````bash
repo sync
```

````bash
Dont build now
